**MultiScope** is a high-performance macro designed for monitoring Sols RNG biome changes across multiple Roblox accounts. It is built using a custom-developed performance engine to be as fast and light-weight as possible, beating previous versions by over 50x in terms of performance.

---

## Key Features

### High-Speed Performance
Powered by a proprietary performance engine, MultiScope monitors biome changes with minimal latency and ultra-low system impact. It is designed to be stable, efficient, and discreet while you do other things on your computer.

### Multi-Account Support
Track multiple accounts simultaneously. Nothing else to say here.

### Anti-AFK and Stealth Tools
Prevents auto kicks due to inactivity without you even knowing.
- **Hidden Mode:** Sends Roblox to the background while keeping it active.

### Clean Dashboard and Interface
A clean interface to manage your configuration.
- **Live Statistics:** Track total event counts and session uptime.
- **Notification Management:** Easily configure and test your endpoints.
- **Customizable Keybinds:** Toggle monitoring on and off with a single hotkey.

## Privacy and Security

MultiScope is built with privacy in mind.
- **Local Operation:** All tasks occur locally on your machine. Only fetching updates and data are done through the internet.
- **Privacy First:** No sensitive credentials or personal data are collected or stored.
